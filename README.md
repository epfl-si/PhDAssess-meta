# PhD Assess Meta

Provide the associated TypeScript types to PhDAssess consumers.

## Install the types

> `npm install --save-dev git+https://github.com/epfl-si/PhDAssess-meta`

Then use it in your project like this:
> `const job = zeebeJob as PhDAssessJob`
